# smart-elevator-control-erd-assignment

A time-boxed system design exercise focused on modeling a scalable elevator control system for large buildings.

---

## ⏱️ Overview

This project was completed under a **1-hour constraint** to simulate real interview conditions.

The approach:
1. Quick hand-drawn ERD (~20 mins)
2. Converted into an initial schema
3. Identified gaps and refined the design
4. Finalized a more realistic and scalable ERD
